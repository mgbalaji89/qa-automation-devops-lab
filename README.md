\# QA Automation \& DevOps Lab



\## Overview

This repository contains an end-to-end QA Automation and CI/CD implementation

built from scratch to simulate real-world project workflows.



The goal of this project is to demonstrate hands-on experience in:

\- UI Automation

\- API Automation

\- CI/CD pipelines

\- Dockerized test execution

\- Reporting and failure analysis



\## Tech Stack

\- UI Automation: Playwright (TypeScript)

\- API Testing: Playwright API

\- CI/CD: Jenkins

\- Containerization: Docker \& Docker Compose

\- Reporting: Allure

\- SCM: GitHub



\## Key Features

\- Page Object Model (OOP-based framework)

\- Data-driven testing

\- Custom logging

\- Dockerized execution

\- Jenkins pipeline integration

\- Automated reporting



\## Execution Flow

1\. Code pushed to GitHub

2\. Jenkins pipeline triggered

3\. Tests executed inside Docker containers

4\. Reports generated and published
